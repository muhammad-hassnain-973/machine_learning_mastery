# Resources — Further Practice Datasets

Ready-to-load real-world datasets used for continued SQL practice beyond the initial crash course.

## SQLite-ready / clean relational (good next step, same shape as the practice DB here)
- **Chinook** — music store DB (artists, albums, tracks, invoices, customers). Ships as a ready `.sqlite`/`.db` file.
  https://github.com/lerocha/chinook-database
- **Northwind** — classic orders/products/employees sample DB.
  https://github.com/jpwhite3/northwind-SQLite3

## Real-world / messier (ML feature engineering practice)
- **Olist Brazilian E-Commerce** (Kaggle) — orders, customers, payments, reviews, geolocation across multiple CSVs. Real-world analogue of the RFM capstone in this repo, at much larger scale.
  https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
- **Credit Card Fraud Detection** (Kaggle, ULB) — imbalanced classification + SQL aggregation/filtering practice.
  https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- **Kaggle SQL-tagged datasets** (general index)
  https://www.kaggle.com/datasets?tags=16641-SQL

## Large-scale, browser-queryable (no download)
- **Google BigQuery Public Datasets** — NYC Taxi trips, GitHub activity, COVID data, and more. First 1TB/month free.
  https://cloud.google.com/bigquery/public-data
